## 概念

- `TypeScript`是微软开发的一个开源的编程语言，通过在`JavaScript`的基础上添加静态类型定义构建而成
- `TypeScript`通过`TypeScript`编译器或Babel转译为`JavaScript`代码，可运行在任何浏览器，任何操作系统
- `Typescript`是`JavaScript`的超级，遵循最新的`ES5/ES6`规范，`Typescript`扩展了`JavaScript`语法
- 很多的开源项目是基于`Typescript`的，学习`Typescript`有助于阅读源码
- `Typescript`提供的类型系统可以帮助我们在写代码时候提供更丰富的语法提示
- 在创建前的编译阶段经过类型检测，可以规避掉很多类型的错误

## 安装和编译
安装
```bash
# 安装
npm i typescript -g

# 查看版本号
tsc -v
````
编译
```bash
# 创建ts文件helloworld
touch helloworld.ts
# 编译helloworld
tsc helloworld.ts
```
生成配置文件，执行以下代码：
```bash
tsc --init

## 数据类型

布尔类型(boolean)
```typescript
let married: boolean = false;
```

数字类型(number)
```typescript
let age: number = 10;
```字符串类型(string) 
```typescript
let str: string = '哈哈哈哈';
```

数组类型(array)
```typescript
let str: string = '哈哈哈哈';
```

元组类型(tuple)
```typescript
// 在 TypeScript 的基础类型中，元组（ Tuple ）表示一个已知数量和类型的数组
const s: [string, number] = ['sfs', 10]
console.log(s)
```

::: tip 数组和元组区别
数组:
1. 每一项都是同一种类型
2. 没有长度限制
3. 用户表示一个列表

元组: 
1. 每一项可以是不同的类型
2. 有预定义的长度
3. 用于表示一个固定的结构
:::

枚举类型(enum)

- 事先考虑某一个变量的所有的可能的值，尽量用自然语言中的单词表示它的每一个值
- 比如说月份、星期、性别、颜色、单位

普通枚举
```typescript
enum Gender{
  GIRL,
  BOY
}
console.log(`李雷是${Gender.BOY}`);
console.log(`韩梅梅是${Gender.GIRL}`);

enum Week{
  MONDAY=1,
  TUESDAY=2
}
console.log(`今天是星期${Week.MONDAY}`);
```
